# Vinland Supply Co.

Vinland Supply is a dashbnoard for a fictional company based on the anime Vinland Saga, which I like a lot.
Link: https://vinland-supply-react-dashboard.vercel.app/

## Install Instructions

Just run: 

```sh
yarn
```
Or
```sh
npm install
```

Then:
```sh
yarn dev
```

## Stack
<ul>
  <li> React (Vite)
  <li> Typescript
  <li> Styled Components
  <li> React-Table
  <li> Chart.js
  <li> React-Hook-Form
</ul>
